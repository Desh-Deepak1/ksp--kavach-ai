# 🛡️ Project Overview

KSP Kavach AI is an AI-powered tactical intelligence platform designed to assist policing and investigative teams through advanced analytics, conversational intelligence, geospatial analysis, and criminal network exploration.

The system consolidates multiple intelligence workflows into a unified operational dashboard, enabling investigators to analyze incidents, identify hidden relationships, monitor emerging threats, and generate actionable insights.

# ⚙️ Setup and Execution Instructions

## 1. Clone Repository

```bash
git clone <repository-url>
cd ksp-kavach-ai
```

---

## 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend development server will start locally.

---

## 3. Backend Setup

```bash
pip install -r backend/requirements.txt
uvicorn app.main:app --reload
```

Backend API server will start with auto-reload enabled.

---

## 4. Production Build

```bash
cd frontend
npm run build
cd ..
catalyst deploy
```

This generates optimized frontend assets and deploys the complete application to Zoho Catalyst.
